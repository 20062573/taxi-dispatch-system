# taxi-dispatch-system

Customer Features :

Register / Login with secure authentication. 

Request a Ride by providing pickup and drop-off details. 

Track Ride Status during the trip. 

View Ride History, including fare and driver information. 

Driver Features :

Register / Login as a driver. 

Accept Ride Requests from customers. 


If driver will accept ride his total earnings will update by 50 euro


With crud operation there are extra feature for customer and driver different as mentioned above
For login used JWT token and project developed in react as front end express and node for backend and mongodb as database


// 

Frontend (React.js) 

Interactive interface for customers and drivers. 

Uses Axios for API communication and React Router for navigation 
 

Backend (Node.js + Express.js) 

Handles all REST API requests for rides, drivers, customers, and fares. 

Manages business logic and dispatch workflows. 
 

Database (MongoDB + Mongoose) 

Stores driver profiles, customer details, ride records, and fare calculations.
